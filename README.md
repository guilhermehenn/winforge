# WinForge

WinForge é uma ferramenta para Windows que centraliza instalação de softwares, manutenção, limpeza e diagnósticos do sistema em um menu simples e prático.

## Download

Baixe a versão mais recente pela página oficial de releases:

[Baixar WinForge](https://github.com/guilhermehenn/WinForge/releases/latest/download/WinForge.zip)

Ou acesse todas as versões publicadas:

[Ver releases](https://github.com/guilhermehenn/WinForge/releases)

## Requisitos

- Windows 10 ou Windows 11
- PowerShell
- Winget instalado
- Permissão de administrador para executar algumas funções

## Como usar

1. Baixe o arquivo `WinForge.zip` pelo link acima.
2. Clique com o botão direito no arquivo ZIP baixado.
3. Vá em **Propriedades**.
4. Marque a opção **Desbloquear**, se ela aparecer.
5. Clique em **Aplicar** e depois em **OK**.
6. Extraia o ZIP.
7. Execute `WinForge.cmd` como administrador.

## Observação sobre bloqueio do Windows

Arquivos baixados da internet podem ser bloqueados pelo Windows por segurança.
